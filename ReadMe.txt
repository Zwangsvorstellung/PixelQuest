Pixel Quest

--------------------------

Incarnez-vous dans la peau d’un blob ou d’un lapin avec comme objectif le cumul de points en tuant des monstres.
Pour ce faire, baladez vous de map en map dans un donjon, et faites ainsi face à de nouveaux adversaires féroces en voulant à votre vie.


Bon jeu et bon score !


-------------
Langage : Lua
Moteur : Love 2D
Ressources graphiques : par NGM/Clove : GameDesigner/développeuse pour le jeu Slayers Online.
Ressources de base réalisées pour le MMORPG Slayers Online.
Musique : Création de JoyBoy 
https://soundcloud.com/allu-ren/crystal-wave-premix

Touche de déplacement jeu et menu :
right, left, up, down
Attaquer : space
Utiliser un objet de l'inventaire

Slot 1 : q
Slot 2 : s
Slot 3 : d

Parler au PNJ : space
Valider une action : return (touche entrée)
Menu Pause : rshift
